## Tech Stack

**Client:** ReactJs, TailwindCSS

**Server:** Node, Express


## Installation

Install apotik-app with npm

```bash
  git clone https://github.com/IMarcellinus/apotik
  cd apotik
  npm install
```

Running User Seeder
```bash
  npx sequelize-cli db:seed:all 
```
